1º coisa: 
Quando criar uma arquivo .dart, tem que criar com letra minuscula
exemplos:
 login.dart
 regoister.dart

 2º coisa:
 Quando criar uma CLASSE, tem que criar com a primeira letra maiuscula.
 Exemplos:
    class Animal{}
    class Login extends Statefull{}
    class Teste extends Stateless{}

Esse é o padrão.
